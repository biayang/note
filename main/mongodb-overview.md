# mongodb 概述

## 版本

- 0.x 起步（2008）
- 1.x 支持复制集和分片集（2010）
- 2.x 更丰富的数据库功能（2012）
- 3.x WiredTiger和周边生态环境（2014）
- 4.x 分布式事务支持（2018）

## 对比SQL

|            | MongoDB                   | RDBMS                |
| ---------- | ------------------------- |
| 数据模型   | 文档模型                  | 关系模型             |
| 数据库类型 | OLTP                      | OLTP                 |
| CURL类型   | nosql                     | sql                  |
| 高可用     | 复制集                    | 集群模式             |
| 横向扩展   | 原生分片                  | 数据分区or应用侵入式 |
| 索引       | B-、全文、地理、多键、TTL | B tree               |
| 开发难度   | 简单                      | 复杂                 |
| 数据容量   | 无上限                    | 千万、亿             |
| 扩展方式   | 垂直+横向                 | 横向                 |
